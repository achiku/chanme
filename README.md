# chanme

Small Blockchain PoC Written in Go for Educational Purpose

- https://github.com/lhartikk/naivechain


#### run server
```
wbs -c wbs.toml
```

#### get blocks
```
http http://localhost:8511/blocks
```

#### create block
```
http POST http://localhost:8511/blocks/add data=ahodfaisdfasd
```
